# GPU ストレステスト

これは、GPU に高い負荷をかけるための単一ファイル WebGL ストレステストページです。

## 機能

- フルスクリーンのアニメーション描画
- 重いフラグメントシェーダーによる高負荷処理
- フィードバック用のオフスクリーンバッファ
- 複数 WebGL コンテキストの並列実行
- FPS、フレーム時間、内部解像度、状態表示

## 使い方

1. `gpu-stress-test.html` を最新のデスクトップブラウザで開きます。
2. `Start Test` を押して負荷を開始します。
3. 各コントロールで負荷を調整します。
4. `Pause` または `Stop and Reset` で負荷を下げます。

## コントロール

- `Resolution Scale`: 内部レンダリング解像度を調整します。
- `Shader Complexity`: フラグメントシェーダーの処理量を調整します。
- `Draw Passes`: 各エンジンが行うオフスクリーン描画回数を調整します。
- `Concurrent Engines`: 複数の WebGL コンテキストを並列に動かします。
- `Speed`: アニメーション速度と時間変化を調整します。

## プリセット

- `Safe`: 最も軽い設定です。
- `Medium`: 標準的な設定です。
- `Extreme`: かなり重い設定です。
- `Ultra`: もっとも重い内蔵設定です。

## 注意

- 負荷が高いため、ファンの回転数が上がる場合があります。
- ブラウザが重くなったら、まず `Pause` を押してください。
- WebGL 対応が必要です。
