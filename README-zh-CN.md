# GPU 压力测试

English: [README.md](https://github.com/a742987/Web-based-GPU-test/blob/main/README.md) | 日本語: [README-ja.md](https://github.com/a742987/Web-based-GPU-test/blob/main/README-ja.md)

这是一个单文件的浏览器 GPU 压力测试页面，右侧使用 Three.js 线框场景，左侧提供多语言控制面板。

## 功能

- 右侧显示 Three.js 3D 线框图形
- 后台使用 WebGL 负载生成 GPU 压力
- 支持简体中文、英文、日语三种语言
- 显示 FPS、帧耗时、内部渲染分辨率和状态

## 使用方法

1. 用现代桌面浏览器打开 `gpu-stress-test.html`。
2. 在 `Language` 里选择显示语言。
3. 点击 `开始测试` 启动或暂停压测。
4. 使用预设和滑杆调整负载。

## 控件说明

- `Language`: 切换界面语言。
- `预设`: 快速切换负载档位。
- `渲染分辨率倍率`: 调整内部渲染分辨率。
- `着色器复杂度`: 调整片元着色器计算量。
- `每帧绘制次数`: 调整离屏绘制轮数。
- `并发引擎`: 调整并行 WebGL 引擎数量。
- `动画速度`: 调整动画速度和时间变化。

## 预设

- `安全`: 最低负载。
- `中等`: 平衡负载。
- `极限`: 高负载。
- `超压`: 最重的内置负载。

## 注意事项

- 这个页面会明显提高 GPU 负载，设备温度和风扇转速可能上升。
- 界面已经尽量压缩成更紧凑的控制布局，便于查看和操作。
- 需要浏览器支持 WebGL。
