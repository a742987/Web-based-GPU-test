# GPU 压力测试

这是一个单文件 WebGL 压力测试页面，用来尽可能提高 GPU 负载。

## 功能

- 全屏动画渲染
- 高复杂度片元着色器
- 离屏反馈缓冲区
- 多个 WebGL 上下文并发执行
- FPS、帧耗时、内部渲染分辨率和状态显示

## 使用方法

1. 用现代桌面浏览器打开 `gpu-stress-test.html`。
2. 点击 `开始测试` 启动压测。
3. 根据需要调整各项参数。
4. 点击 `暂停` 或 `停止并重置` 降低负载。

## 控件说明

- `渲染分辨率倍率`: 调整内部渲染分辨率，越高越吃性能。
- `着色器复杂度`: 控制片元着色器的循环和计算量。
- `每帧绘制次数`: 控制每个引擎每帧执行的离屏轮数。
- `并发引擎`: 同时运行多个 WebGL 上下文。
- `动画速度`: 调整动画速度和时间变化幅度。

## 预设

- `安全`: 最低负载，适合确认页面正常工作。
- `中等`: 平衡负载。
- `极限`: 高负载。
- `超压`: 默认更重的高负载设置。

## 注意事项

- 这个页面可能会明显提高风扇转速并增加设备温度。
- 如果浏览器变慢，先点 `暂停`，再点 `停止并重置`。
- 需要浏览器支持 WebGL。
